System Documentation

Technical details and methodologies behind ScamGuard AI

Eng. Abdessamad Bourkibate

Developer & Researcher

Academic research project developing a multi-mode AI system for Arabic text threat detection. Combines machine learning (Linear SVM) with advanced language models (BERT - CAMeL-Lab) for 99.35% accuracy content analysis.

Processing Pipeline

1. Arabic text cleaning
2. TF-IDF extraction (10K features)
3. 6 custom semantic features
4. Feature vector merging
5. SVM + BERT classification

Arabic NLP

• Alif/Ta/Ya normalization
• Diacritics removal
• Stopwords filtering
• Account number encoding
• Bigram TF-IDF (1,2)

Linear SVM Model

• class_weight = 'balanced'
• Accuracy: 99.35%
• Precision: 96—100%
• Recall: 94—100%
• Test set: 41,990 samples

BERT Arabic Model

• CAMeL-Lab (NYU Abu Dhabi)
• bert-base-arabic-camelbert-da
• Sentiment (pos/neg/neutral)
• 4 custom emotion metrics
• Up to 512 token processing

Training Data

• 206,849 real tweets (98.5%)
• 3,102 synthetic samples (1.5%)
• Scraped: 20.7M tweets
• Source: Twitter/Kaggle Arabic
• 3 balanced categories

Advanced Detection

• 140+ threat patterns
• 6 threat categories
• 4 analysis modes
• Printable HTML reports
• Arabic + English UI
